Création d’une maquette

Création d’une maquette qui prend en compte les spécificités fonctionnelles décrites dans les cas d’utilisations et les User Story. Il faudra aussi présenter votre projet en suivant les modèles de la méthode Merise :

  • Modèle conceptuel des données (MCD),
  • Modèle logique des données (MLD) et
  • Modèle physique des données (MPD).

⇒ Démarche perso à faire

Sera évaluée automatiquement en présentant votre projet en suivant les points principaux : - Code documenté, une documentation sur l’installation est dispo etc - Respect des normes, développement sécurisé (pas de failles/fuites) - Utilisation d’un outil collaboratif

Exemples de projets : ASCII-Art-Web, Groupie-tracker, Forum
