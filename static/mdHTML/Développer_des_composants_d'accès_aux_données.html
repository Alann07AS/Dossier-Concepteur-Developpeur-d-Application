Social Network

Social Network est une app multicouche. Il est composer de plusieur API: * Un serveur node d’image * Un serveur node de chat * Un serveur node React pour le client * Un serveur Golang pour les fonctionalitées

Il y a egallement un second serveur chat en Golang qui n’est pas utiliser pour le deploiment global du projet.
